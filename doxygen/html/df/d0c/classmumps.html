Detailed Description

Solver for general sparse matrices.

It solves the system of linear equations A*X=B with a general sparse matrix A. The RHS matrix B is a dense matrix.

The solver automatically detects whether A is distributed on many processes or centralized on the root process. The matrix A should be stored in the Compressed Sparse Row (CSR) format with one-based indexing.

To set control parameters, use the overloaded function call operator, which allows access to the icntl array.

auto solver = mumps<double, int_t>();
solver(3) = 0; // suppress output
